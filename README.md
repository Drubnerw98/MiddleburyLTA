# MiddleburyLTA

Public-facing website for the Middlebury Taxpayers Association.  
Built with Next.js and Firebase, it explains the groupâ€™s mission and includes an interactive tax impact calculator.

## Features

- **Landing page** with overview of the organization and contact info
- **Tax Impact Calculator** that estimates property tax changes based on assessment and mill rate inputs
- **Articles & Links** section for news, resources, and external references
- **Admin tools** (auth-protected) to manage posts and links
- Responsive layout designed for desktop and mobile

## Tech Stack

- **Frontend:** Next.js (React, TypeScript)
- **Styling:** CSS / Tailwind
- **Backend / Data:** Firebase (Firestore, Auth)
- **Deployment:** Vercel
